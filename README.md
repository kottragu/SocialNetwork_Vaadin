# Simple socialnetwork #
## Features ##
* Create user by nickname
* Find others users by nickname
* Send messages
* Change nickname or password (also, you can reset it)
* A few jokes on main page

## Techological stack ##
* Java 11
* Vaadin 14.4
* Maven
* Spring boot 2.4
* Spring Data JPA
* Spring Security
* Project Lombok
* MySQL
